# Docusaurus Plugin

If you're using [Docusaurus](https://docusaurus.io/), we have a dedicated npm package that brings the AI chat experience to your docs.

## Installation

```bash
npm install vetradocs-docusaurus
```

## Quick Start

### 1. Add to your Theme

You need to wrap your site with the `VetradocsChat` and `VetradocsFloatingBar` components. The easiest way is to Swizzle the `Layout` or `Root`.

For example, using `src/theme/Root.js`:

```jsx
import React from 'react';
import { VetradocsChat, VetradocsFloatingBar } from 'vetradocs-docusaurus';
import 'vetradocs-docusaurus/dist/theme/VetradocsChat/styles.css';
import 'vetradocs-docusaurus/dist/theme/VetradocsFloatingBar/styles.css';

export default function Root({children}) {
  return (
    <>
      {children}
      <VetradocsFloatingBar />
      <VetradocsChat />
    </>
  );
}
```

### 2. Build the Search Index

Add a script to your `package.json` to build the search index after your documentation is built.

```json
"scripts": {
  "build": "docusaurus build && npx vetradocs-build-docusaurus"
}
```

This will generate `static/search-index.json`.

### 3. Backend Setup

You strictly need a backend service to handle AI requests. We provide a CLI to handle this.

1.  Run the backend creator:
    ```bash
    npx create-vetradocs-backend@latest
    ```
2.  Choose **Cloudflare Workers** (Recommended) or Node.js.
3.  Deploy and get your URL and API Key.

See [Backend Setup](./backend-setup) for a step-by-step guide.

Once set up, update your configuration in your Docusaurus theme:

```jsx
<VetradocsChat 
    apiEndpoint="https://your-backend.com"
    apiKey="your-secret-api-key"
/>
```

## Configuration

Pass props to the components to configure them.

```jsx
<VetradocsFloatingBar 
    apiEndpoint="/api/your-chat-endpoint" 
    title="My AI Docs" 
    accentColor="#fa5252" 
/>
<VetradocsChat 
    apiEndpoint="/api/your-chat-endpoint"
/>
```

| Prop | Default | Description |
|------|---------|-------------|
| `apiEndpoint` | `/api/chat` | Endpoint for chat API |
| `indexPath` | `/search-index.json` | Path to generated index |
| `accentColor` | `#f97316` | Theme color |
| `title` | `AI Assistant` | Title in chat header |
